موزّع حصة النشاط — النسخة الأولية التنفيذية

التشغيل:
1) فك ضغط الملف.
2) افتح index.html باستخدام Google Chrome أو Microsoft Edge.
3) تبدأ البيانات فارغة وتحفظ تلقائيًا داخل المتصفح.

الوظائف المنفذة:
- إعداد اسم المدرسة والجنس والمراحل ونوع التعليم.
- توليد الصفوف والشعب تلقائيًا.
- تحديد ضوابط 35 حصة، نصاب 24، و36 أسبوعًا.
- استيراد المعلمين من Excel أو CSV أو نص مباشر.
- مراجعة الاسم والتخصص والنصاب قبل الاعتماد.
- تصنيف المواد والفصول والشعب لكل معلم.
- بناء الإسنادات وعدد الحصص الأسبوعية.
- حساب الحصص السنوية والحد الأعلى الآمن لنسبة 10%.
- التحقق من النصاب والأهلية.
- إنشاء توزيع أسبوعي لحصة النشاط لكل معلم مؤهل.
- حفظ تلقائي محلي باستخدام IndexedDB مع بديل Local Storage.
- إنشاء واستعادة نسخة احتياطية JSON.

ملاحظة:
- قراءة ملفات Excel تعتمد على مكتبة SheetJS المحملة من الإنترنت.
- عند عدم الاتصال، استخدم CSV أو الاستيراد النصي.
- الطباعة والتوقيع لم يضافا بعد حسب الاتفاق، وسيتم بناؤهما بعد تثبيت منطق البيانات والتوزيع.


تحديث الواجهة:
- يمكن طي القائمة الجانبية وفتحها من الزر أعلى القائمة.
- يتذكر المتصفح حالة القائمة بين مرات التشغيل.
- الخط الأساسي في الواجهة هو Cairo، ويُحمّل من Google Fonts عند توفر الإنترنت، مع خطوط بديلة عند العمل دون اتصال.
